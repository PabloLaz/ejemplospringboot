# Proyecto de ejemplo Springboot EOI

Este proyecto tiene como objetivo crear un modelo de aplicación guía para el aprendizaje de springboot.




## Software utilizado

- SpringBoot 3.0.7 
- Maven
- JAR Packaging
- Java 20

## Entorno de desarrollo

- IntelliJIDEA

### Plugins del entorno de desarrollo

- SonarLint
- JPA Buddy
- Database Navigator
